# 虚空创痕

## 本书信息

本书译自 *VoidScarred*，原作者 Mike Brooks，版权归 Games Workshop 及 the Black Library 所有。  

本书为免费电子发布的不盈利民间译本，节译者及说书人维克特 Vectaidh。  

原文无章节名，说书章节名系说书人自行添加。 说书内容据说书人理解进行取舍，内容有所概括总结。

如有疑问欢迎纠错。
